# Credit Union Loan Default Prediction

This project builds a machine learning model to predict whether a borrower will default on a loan using financial and credit-related attributes.

Predicting loan default risk is important for financial institutions such as credit unions because it helps reduce financial losses and improve lending decisions.

## Project Workflow

- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Logistic Regression Model
- Random Forest Model
- Model Evaluation using Accuracy and Confusion Matrix

## Results

The Logistic Regression model achieved approximately **88% accuracy** on the test dataset.
The Random Forest model produced similar performance when predicting loan default risk.
